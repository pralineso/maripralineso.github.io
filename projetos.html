Projetos

Nessa página serão listados todos os projetos que ja foram desenvolvidos mim ao longo dos anos.

PitStop

O projeto PitStop foi um aplicativo desenvolvido durante a faculdade na disciplina de Engenharia de Software em 2018. Esse aplicativo tem como objetivo auxiliar as pessoas a gerenciar o seus automóveis.

SCAS

O projeto SCAS foi um sistema web desenvolvido durante a faculdade no estágio supervisionado para o Laboratório de Análise de Sementes do Acre (LASFAC) em 2019, para o controle do processo de análise de sementes do laboratório.

Lembretinhos

O projeto Lembretinhos é um aplicativo para lembrar aos seus usuários de seus eventos. O aplicativo Lembretinhos foi idealizado a muito tempo e somente em janeiro de 2020 "saiu do papel", o Lembretinhos ainda esta em fase de desenvolvimento.

App-gatinhos

O projeto App-gatinhos é uma aplicação que foi desenvolvida durante o curso de DataScience (que estou fazendo atualmente). Esssa aplicação faz uma breve análise dos dados das raças de alguns gatos.
